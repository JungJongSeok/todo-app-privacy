투두리스트 개인정보처리방침

시행일: 2026년 4월 14일

1. 개인정보의 수집 및 이용

투두리스트 앱은 사용자의 개인정보를 수집하지 않습니다. 작성한 할 일 데이터는 사용자의 기기에만 저장되며 외부 서버로 전송되지 않습니다.

2. 데이터 저장

할 일 데이터는 기기의 로컬 저장소(MMKV)에만 저장됩니다. 계정/로그인 불필요. 앱 삭제 시 모든 데이터가 삭제됩니다.

3. 제3자 제공

사용자 정보를 제3자에게 제공하지 않습니다.

4. 광고 및 분석

광고 SDK나 분석 도구를 사용하지 않습니다.

Todo data is stored only in local MMKV storage. No account/login required. All data deleted on uninstall.
